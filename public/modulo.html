％演算子

C言語の特徴で、int どうしの演算の結果は int になる。 short どうしの演算は short。

小数を知らなかった低学年時代、「10割る3」の答えは「3あまり1」だった。 C言語の割り算もこれ。

  int x=10;
  int y=3;
  int z;
  int w;

  z = x / y; // 3 が代入される。
  w = x % y; // 1 が代入される。

% で割り算のあまりを求めることは、C 言語のプログラミングでは 受講生が予想するよりもたくさん出てくる。

演習問題

関数 foward() と stop() がある。

  void forward(void)
  {
  	Mtr_Run_lv(30000,-30000,0,0,0,0);
  	Wait(50);
  }
  void stop(void)
  {
  	Mtr_Run_lv(0,0,0,0,0,0);
  	Wait(50);
  }

これら関数に手を加えることなく、 低速、中速、高速の三つの走行をする low(), middle(), high() をプログラムせよ。 できたらそれが PWM だ。 2年前の資料だが、 http://edu.melt.kyutech.ac.jp/2010/robocar/pwm/参照。 速度を切り返すとき、LED の点灯が変わるとわかりやすい。
