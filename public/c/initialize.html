変数の初期化

C言語の変数定義は「記憶領域の確保」しかしない。 確保した領域がどんなビットパターンで埋められているか、予期できない。 

// 親切なコンパイラだと、エラーになる。
#include <stdio.h>
int main(void)
{
	int x;
	printf("x=%i\n",x);
	return 0;
}
  • 定義と同時に初期化
    #include <stdio.h>
int main(void)
{
	int x=1;

	printf("x=%i\n",x);
	return 0;
}
  • 使う直前に初期化
    #include <stdio.h>
int main(void)
{
	int x;

	x=1;
	printf("x=%i\n",x);
	return 0;
}
