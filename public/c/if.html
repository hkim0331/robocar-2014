if

if 文の正当な形はelse 以下を省略しないこれ;

if (condition) {
	true case;
} else {
	false case;
}
  • else { } は省略できる。 しかし、Haskell などの現代的な言語では省略できない。 それは意味があること。みんなもしばらく Haskell にならってほしいが、、、
  • if は入れ子にできる。 入れ子にできるのは true case の部分と false case の部分の2箇所ある。
    • if (condition) {
	true case;
} else { if (conditon 2) {
		true case 2;
	} else {
		false case 2;
	}
}
    上のようにプログラムすると見にくいので、下のように書こう。else if チェーンと呼ばれる。 
    if (condition) {
	true case;
} else if (conditon 2) {
	true case 2;
} else {
	false case 2;
}
