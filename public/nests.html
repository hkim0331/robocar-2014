入れ子

for 文の内側に for 文、 if 文の内側に if 文が来るようなプログラムの構造を「入れ子」という。

例題:九九の表

九九の表は 1の段、2の段,,,があり、各段も かける1, かける2,,,という繰り返しだ。そいつをプリントするプログラムは基本的に入れ子になる。

void kuku()
{
	int i;
	int j;

	for (j=1; j<10; j+=1) { // 外側の for
		for (i=1; i<10; i+=1) { // 内側の for
			printf("\t%i",i*j);
		}
		printf("\n");
	}
}

関数呼び出しを入れ子に

関数呼び出しは入れ子にできる。

#include <stdio.h>
int add1(int x)
{
	return x+1;
}

int doubles(int x)
{
	return x*2;
}

int sq(int x)
{
	return x*x;
}

int main(void)
{
	int x=2;

	printf("%i\n",sq(dobles(add1(x))));
	return 0;
}

(C言語では)関数定義は入れ子にできない

呼び出しと定義を区別すること。

/* こういうのは C 言語では不可。*/
int add1(int x)
{
	// add1() の定義ブロックの内側で新たな関数の定義は反則。
	int doubles(int x)
	{
		return x*2;
	}
	return doubles(x+1);
}
