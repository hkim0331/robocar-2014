もっと配列

配列を使ったことをもっと利用する。配列とループは相性抜群。 注意: 以下のプログラムは最近の C でないとコンパイルできない。 古いのだとエラーで動かない。そんときはおとなしく int b[5] と定義しよう。 関数名も戻さないと。

void stop_if_black(int n)
{
	int b[n];
	int i;

	for (i=0; i<n; i++) {
		step(1);
		if (both_black()) {
			b[i]=1;
		} else {
			b[i]=0;
		}
	}

	if (b[0]==1 && b[1]==1 && ...) {
		buzzer();
		stop();
	}
}
