ブザーを鳴らす

  • void BuzzerSet(unsigned char pitch, unsigned char vol)
    • pitch:音のピッチを設定する。設定できる値は 0～255 で、値が大きいほど音程は低くなる。
    • vol: ボリュームを設定する。設定できる値は 0～128。値が大きいほど音量は大きくなる。
  • void BuzzerStart(void)

    設定にしたがい、ブザーを鳴らす。

  • void BuzzerStop(void)

    ブザーを止める。

  • unsigned char isBuzzer(void)

    ブザーが鳴っていたら1、そうでなければ 0 を返す。


void buzzer(void)
{
        BuzzerSet(100,100);
	BuzzerStart();
	Wait(1000);
	BuzzerStop();
	Wait(1000);
}

int main(void)
{
	const unsigned short MainCycle = 60;
	Init(MainCycle);


	buzzer();

	return 0;
}

BuzzerSet(pitch, vol) の pitch に音の高さを指定できる。
