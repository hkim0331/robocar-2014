ちょっとだけ回答

以下が100点とは思わないこと。 自分で考えず書き写すだけはまったく勉強にならない。 そんなのより、自分の力で考えたようにプログラムしロボカーが期待通りに動かなかったとき、 自分の考えのどこに不足があったか、 考えをC言語に翻訳する過程のどこにミスが入り込んだか、 C言語の何を自分は理解していないか、 汗かいて考えよう。それが単位や成績よりも大切なことだ。

手元にある関数 step(), stop(), both_blacks() を有効利用する。

  1. step(1)を20回読んでロボカーを 20cm 進ませる。
  2. step(1)の直後にセンサの値をboth_blacks() で計測し、配列 bw[i] に記録する。
  3. 動作完了後、配列 bw[i] の値をLEDで表示する。 bw[i]が１つまり黒を感知していればLED(1)、bw[0]が0つまり白を感知していれば LED(0)の値を１秒間キープ。
int counts(int n)
{
	int i;
	int bw[20]; // black==1, white ==0

	// 箇条書きの１番を実装する部分
	for (i=0; i < n; i++) {
		step(1);
		bw[i] = both_blacks();//２番の実装
	}
	stop();

	// ３番の実装。
	for (i=0; i < n; i++) {
		LED(0);
		Wait(1000);
		LED(bw[i]);//
		Wait(1000);
	}
}
