プロジェクトの始め方

ロボカープログラミングを進めるためのプロジェクトのひな形を作った。 LED、モーターなどのテーマごとにこのひな形をコピーし、利用するとよい。

  1. プロジェクトのひな形 robocar.zip をクリックしてダウンロード、 適当な場所にセーブする。 情報センターの firefox は「ダウンロード」フォルダの下に robocar.zip をセーブするはず。
  2. LPCXpresso を立ち上げ、左下方のペインから Import projects を選ぶ。

  3. browse ボタンを押し、

  4. 先ほどダウンロードした robocar.zip を選択し、open、そして next。

  5. robocar(robocar) が選択されていることを確認して、finish。

  6. ウィンドウ左上ペインに現れる robocar を右クリック、Rename を選択して、 プロジェクト名を変えておく。

アドバイス

  • robocar(robocar)が選択できない場合、 見えない robocar プロジェクトができている。 

    $ rm -r ~/robocar/workspace/robocar

    を実行し、見えない robocar を消してから再度やってみるといい。

  • 最後の Rename は今後、複数のプロジェクトを作るために必要です。
