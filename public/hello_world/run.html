hello_world.c を以下のように書き換える。

#include <stdio.h>
#include <stdlib.h>

void run(void)
{
	puts("!!!Hello World!!!");
}

int main(void)
{
	run();
	return EXIT_SUCCESS;
}
  • 関数 puts() の情報は stdio.h に定義されている。 #include <stdio.h> の行をコメントアウトしてビルドしたとき、 何が起こるか。実験せよ。

  • EXIT_SUCCESS の情報は stdlib.h に定義されている。 #include <stdlib.h> の行をコメントアウトしてビルドしたとき、 何が起こるか。確認せよ。

  • 原則として、main 関数の戻り値は int と決められている。 void main(void)と書き換え、return EXIT_SUCCESS; をコメントアウトし、 ビルドしたとき何が起こるか？確認せよ。

!!!Hello World!!! を 100回

コピーペースト100回はめんどうだし、融通がきかない。

プログラムをコピーペーストする代わりに、 「100回繰り返せ」をプログラムする。

void run(void)
{
	int i;

	for (i=0; i < 100; i++) {
		puts("!!!Hello World!!!");
	}
}

int main(void)
{
	run();
	return EXIT_SUCCESS;
}

!!!Hello World!!! を n 回

繰り返しの回数をrun()の外部から与える。

void run(int n)
{
	int i;

	for (i=0; i < n; i++) {
		puts("!!!Hello World!!!");
	}
}

int main(void)
{
	run(100);
	return EXIT_SUCCESS;
}

どこがどう変わったかよーく見ること。

forward, turn

次のプログラムは不完全である。関数 run() から呼び出すべき 関数forward() と関数 turn()を定義していない。

しかし、これまでの授業を集中して受講していた人にはぼんやり、 あるいは、はっきりとわかるはず。 これは前週の課題の模範解答のひな形です。


void run(int n, int l)
{
	int i;

	for (i=0; i < n; i++) {
		forward(l);
		turn(360/n);
	}
}

int main(void)
{
	run(4,40);
	return 0;
}

演習問題

上のプログラムを完成させなさい。わからない受講生のために書くと、 void forward(int n), void turn(int n, int l) をきちんと定義しろってこと。 ビルド、焼き付けたら、ロボカーは 一辺 40cm の正方形のエッジ上を動くはず。 run() の引き数を変えると、三角走りや五角形走りもする。
